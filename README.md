# TFSAgent  

Treasure From Sxxt [粪海狂蛆]  

Agent for academic direction monitoring    

# 定时从arxiv上抓取论文处理后上传notion database  

基于arxiv官方api实现  

api目前不提供日期级别精确搜索，因此每天检索固定量文献并剔除重复项


在项目根目录下创建.env          
OPENAI_API_KEY=  
OPENAI_BASE_URL=  
NOTION_TOKEN=  
NOTION_DATABASE_ID=  
notion服务器连接参考谷歌
